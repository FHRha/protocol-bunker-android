# Shared Contract

Локальные контрактные материалы для Android host версии.

- `contracts.ts` - типы фактических HTTP-ответов `server-go`.
- `sample-responses.json` - примеры JSON для smoke/ручной проверки.

Содержимое синхронизировано с текущими endpoint:

- `GET /health`
- `GET /api/scenarios`
- `POST /api/overlay-links`
- `GET /overlay-control/state`
- `POST /overlay-control/save`
- `POST /overlay-control/action`
